# Exercício: Filtragem de Nomes com Lambdas e Streams

Projeto desenvolvido como parte dos exercícios do curso da EBAC (Escola Britânica de Artes Criativas e Tecnologia). O objetivo é praticar o uso de **Interfaces Funcionais**, **Expressões Lambda** e a **API de Streams** do Java.

##  Descrição da Atividade

O desafio consiste em criar uma aplicação que:
1.  Contém uma lista de nomes (ex: "Ana", "Bruno", "Carlos", "Amanda", "Beatriz").
2.  Utiliza uma **Interface Funcional customizada** chamada `FiltroNome`.
3.  Usa Expressões Lambda para filtrar e exibir no console apenas os nomes que começam com a letra **"A"**.

##  Tecnologias Utilizadas

* **Java** (versão 8 ou superior)
* **Paradigma Funcional** (Lambdas, Functional Interfaces)
* **Java Streams API**

##  Estrutura do Projeto

O código está organizado da seguinte forma:

* `src/util`: Contém a interface funcional `FiltroNome`.
* `src/application`: Contém a classe `Principal` com o método `main`, onde a lista é processada.
